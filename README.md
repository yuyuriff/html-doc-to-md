# Генератор Markdown из HTML документации OpenNLP

## Что делает

Скачивает страницу `https://opennlp.apache.org/docs/2.5.7/manual/opennlp.html` и конвертирует её в Markdown

Создаёт директорию `assets/` и скачивает в неё изображения

Выводит в консоль статус выполнения: успех / фатальные ошибки / предупреждения (например, если не удалось скачать изображение)

Результат сохраняется в `opennlp_manual.md`

## Как запустить

```bash
./gradlew run
```

## Примечание

Конвертация рассчитана на структуру конкретной страницы OpenNLP manual (подробнее в комментариях к коду)
